# testdubbodemo
整合springMvc+dubbo+zookeeper的一个小demo，纯粹用于学习和记录进度  

总共用到了2台服务器，一台用于搭建zookeeper+dubbo-admin，成为注册中心，另一台上通过maven插件用两个端口分别运行Controller和Service。
